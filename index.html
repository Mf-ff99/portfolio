Moodify

  • Deployment
  • GitHub

Moodify is a mood and sleep tracking application designed to meet the needs of those with a desire to improve their mental health. Moodify keeps track of entries for hours slept, mood, as well as journal entries for building a comprehensive database of your mental health. Moodify was created from scratch using ReactJS, vanilla CSS, JavaScript, Express, and Node.js.

LearnSpanish!

  • Deployment
  • GitHub

LearnSpanish! was created from scratch using ReactJS, vanilla CSS, JavaScript, Express, and Node.js. LearnSpanish! uses a spaced-repetiton system to teach users Spanish. Utilizing a linked list on the backend, I was able to update the database with the user's word score, and correctly get the correct word from the database.

Coming soon is the ability for users to add their own cards, as well as choose their own language to study. I built this as a prototype for my own Android/iOS app to fulfill the needs of language-learners in a simple yet productive environment. This is just the start of one of my dream projects, more to come soon.
